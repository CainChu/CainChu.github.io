如何同步github內容

之前大家應該已經申請了你自己的github帳號，也建立了自己的網站專案
例如我的帳號是 CainChu ，所以專案名稱應該是 CainChu.github.io

打開你的網站專案資料夾

  1. 首先打開你的 github 同步軟體，並選擇你的網站專案

  2. 你可以在你的傳案選項中按下滑鼠右鍵，選擇 Open in Explorer ，打開你之前設定的同步資料夾位置(我的是PC版，我想MAC應該不會差太多)

3.更新你的資料夾 (放上的新的網頁，或者修改的一些js...)

同步上傳到github

  1. 打開你的 github 同步軟體，選擇你的專案，中間會告知你有那些檔案變動了(不管是新增修改還是被刪除都會顯示)，你可以把確定要同步的檔案打勾，最要的是下方的說明一定得填，填什麼內容不限制，這是將來給自己(或別人)看的，以程式的概念就是告知人家說，你這次的新版本修改的什麼功能，好了之後按下下方的 Commit to master

  2. 這時候還沒有上傳，僅是確認這次修改而已，最後請務必記得按下右上角的 Sync 開始同步上傳，這裡很多人忘記!! (這代表你也可以Commit很多次後再一起Sync)

3.這時候你可以上去你自己的 guthub.com 網站 ，應該就可以看到你自己的更新

網站注意事項

  1. 你的網站位置會是 帳號.github.io (比方CainChu.github.io)
  2. 網站根目錄預設是 index.html ，這代表你的首頁，沒放進去會找不到東西
  3. 瀏覽器會暫存你的舊網頁，如果你確定網站已經更新了但是卻一直看到舊網頁，可以試著清除暫存，可以看教學
